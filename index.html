ATAQUE

Desvendando a Tecnologia: Como o Ataque Funciona

No Asteri, acreditamos na máxima 'a melhor defesa é o ataque', e é por isso que escolhemos um leque como nossa arma principal. Optamos por essa solução devido ao seu alcance impressionante para atingir nossos adversários. O funcionamento é simples e eficaz: quando o sensor é ativado, o leque se abre com o auxílio de um motor, girando 90º para a direita. Para recuar, a peça realiza o mesmo movimento, mas na direção oposta. Utilizamos um canudo e um palito de churrasco para garantir que a arma mova apenas um lado do leque, evitando movimentos indesejados.
HARDWARE

Conheça o Coração do Asteri: Hardware de Ponta

O Asteri é equipado com componentes-chave, como o Arduino Mega para controle e programação, sensores de chaves de fim de curso para detecção, motores de roda com redução para mobilidade precisa, rodízio para adaptação em terrenos variados e pilhas confiáveis para energia. Além disso, possui uma fita de LED endereçável para personalização visual, um shield de morto para segurança e um módulo HC-06 para comunicação sem fio. Com esses elementos, o Asteri é altamente funcional e versátil, pronto para triunfar em missões de combate desafiadoras.
